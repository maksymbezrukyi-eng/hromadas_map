# Інструмент відбору громад

Уніфікований веб-інструмент для відбору та моніторингу громад у програмі «Delivery of universal Early Childhood Development services in primary health care» (UNICEF Ukraine × KfW).

## Використання

Відкрити: `https://YOUR_USERNAME.github.io/hromadas-map`

Три вкладки:
- **◉ Карта** — 68 громад на інтерактивній карті з фільтрами по області та типу KfW
- **▦ Дашборд** — ключові показники, розподіл за KfW, топ областей, розмір громад
- **≡ Таблиця** — сортована таблиця з пошуком та фільтрами

## Структура репозиторію

```
hromadas-map/
├── index.html          # Основний файл застосунку
├── README.md           # Документація та changelog
└── data/               # Дані (додаються в наступних ітераціях)
    └── boundaries.geojson  # Межі громад (v0.2)
```

- Топ-20 громад за зваженим балом (з'являється коли є дані)
- Середні бали по 7 доменах з вагами (з'являється коли є дані)
- Розподіл зважених балів по вибірці (histogram)
- Радарна діаграма середнього профілю вибірки
- Зведена таблиця по областях з колонками short-list, відібрано, середній бал

### v0.3 — 2026-04-10

**Живі дані з Google Sheets**

- Підключено Google Sheets CSV API — карта читає актуальні статуси при кожному завантаженні
- Маркери та межі громад кольоруються за статусом відбору (не за KfW як у v0.1-v0.2)
- Легенда статусів: зелений = short-list/відібрано, помаранчевий = резерв, червоний = виключено, сірий = очікує
- Кнопка ручної синхронізації в бічній панелі
- Автоматичне оновлення кожні 5 хвилин
- Індикатор часу останньої синхронізації
- Таблиця розширена двома колонками: Статус інтерв'ю + Фінальний статус
- Пріоритет кольорів: Фінальний статус → Short-list → KfW (fallback)

**Межі громад**

- Підключено GeoJSON з межами всіх 68 громад (Leaflet GeoJSON layer)
